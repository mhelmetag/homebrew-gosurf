# GoSurf for Homebrew

The official tap for the GoSurf CLI.

## Initial Setup

If you don't have homebrew, install that first.

Then, add this tap:
```
$ brew tap mhelmetag/gosurf
```

## Installing

To install the GoSurf CLI:
```
$ brew install gosurf
```

## Updating
```
$ brew upgrade gosurf
```
